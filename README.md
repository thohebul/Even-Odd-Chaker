# 🔢 Even/Odd Checker

A simple and responsive **Even/Odd Checker** web application built using **HTML, CSS, Bootstrap, and JavaScript**. Users can enter a number and instantly find out whether it is **Even** or **Odd**.

## 📖 How It Works

1. Enter any number in the input field.
2. Click the **"Press Here"** button.
3. The application checks the entered number.
4. It instantly displays whether the number is **Even** or **Odd**.
